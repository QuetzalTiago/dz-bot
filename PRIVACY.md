# Privacy Policy

_Last updated: 2026-07-05_

This document describes the data the bot collects, why, and the controls
available to users. It is a starting template — have it reviewed by legal
counsel before relying on it for a commercial deployment.

## Data we store

| Data | Purpose | Retention |
|------|---------|-----------|
| Discord user ID + accumulated voice-channel time | Powering the `leaderboard` and `status` features | Until erased on request |
| Discord user ID + song request history | Powering `most_played` / `most_requested` | Until erased on request |
| Chess game records from Lichess | Powering `chess_leaderboard` | Until erased on request |

We do **not** store message content, and secrets/tokens are never logged.

## User controls

- `my_data` — shows what the bot has stored about you.
- `forget_me` — permanently erases your stored data (right to erasure).

## Third-party services

The bot calls third-party APIs to provide features (OpenWeather, Coinbase,
API-Sports, Steam, Lichess, Genius, Spotify, and an LLM provider). Requests to
those services are subject to their own privacy policies.

## Contact

For data requests that cannot be self-served, contact the server operator.
